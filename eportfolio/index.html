This is an assignment for a course I take at the University College of Ghent (HoGent), called 'Webapplicaties I' (Webapplications I). It will also be used as a tool for another course called 'iTalent'. All completed assignments for iTalent will be available on its dedicated page.

This website is written in English except for the iTalent page. The reason for this is twofold. Firstly, I've written everything in English because I want it to be internationally accessible, and secondly, because the de facto language in the field of Computer Science is English. The assignments for iTalent will not be translated into English because this is a course given in Dutch and all assignments were to be submitted in Dutch.
