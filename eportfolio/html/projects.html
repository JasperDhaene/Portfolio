Fuzzy rulesystem for a racing simulation

Fuzzy logic was introduced by Lotfi Zadeh in 1965. He proposed a new approach to causative computing, drawing the analogy with humans which usually reason in “fuzzy” terms. This project implements a fuzzy rule-based system using Zadeh's fuzzy logic, including a practical application – the fuzzy car controller. It parses sensorial data and predefined rules to allow a race-car to navigate a simplified race track autonomously.

Written in Java
