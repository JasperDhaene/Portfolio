Vaag regel- systeem voor race simulatie

Vage logica werd ontwikkeld door Lotfi Zadeh in 1965. Hij stelde een nieuwe aanpak voor causative computing voor. Hij haalde zijn inspiratie bij de manier waarop mensen ook denken in "vage" termen. Dit project implementeerd een vaag regelsysteem, gebruikmakende van Zadeh's vage logica, als ook een praktische applicatie - een vage auto controller. Het verwerkt sensor data en voorgedefinieerde regels zorgen ervoor dat de race auto zich autonoom kan navigeren op de virtuele race circuits.

Geschreven in Java
