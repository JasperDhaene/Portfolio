Jasper D'haene

Front-end developer

Hi !

I'm Jasper D'haene, a front-end web developer living in Ghent, Belgium.

I am a creative person who loves to solve problems with code. Eager to learn and self-taught in many things. I often wonder how websites work, what goes on under the hood, or how they look the way they do, so I take a peek and try to learn from it. I am definitely not afraid to dig through some unfamiliar code. In my code, I try to find a balance between abstraction and clarity: clean code with appropriate design patterns but still readable for myself and other developers. That also means I write documentation where the code isn't self-evident.

In my spare time, I read, go to the fitness, make music, or learn a new language.

I like to work on projects that truly help people in new and innovative ways, while being very attentive to the design and the user experience of the project. That's why I was so glad I did my internship at Open Webslides. I worked on a project there that completely changes the way teachers create course material and how students interact with it.

But I'm looking for a new challenge now! I'm looking to work on projects that:

  • help me develop new skills
  • allow me to build the best solution for the user
  • are not just functional, but beautiful as well
